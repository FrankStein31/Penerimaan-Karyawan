APLIKASI PENERIMAAN KARYAWAN LARAVEL PHP >=8.1
- ubah env.example menjadi .env
- buat database pabrikgula dan import pabrikgula.sql
- composer update
- php artisan key:generate
- php artisan storage:link
- php artisan serve
